# Splatoon-style Browser Game

3D-игра в браузере с видом от третьего лица в стиле Splatoon. Цель — закрасить игровую арену цветом своей команды.

## 🎮 Описание

Игра поддерживает 4 команды (красные, синие, зелёные, жёлтые) по 3 игрока в каждой. Вы управляете одним персонажем и должны закрасить как можно большую часть арены краской своей команды.

### Особенности геймплея:

- **Система территорий**: 
  - По своей территории — быстрое передвижение (+50% к скорости)
  - По нейтральной — нормальная скорость
  - По вражеской — медленное передвижение (-50% к скорости)

- **Стрельба краской**: Закрашивайте арену, стреляя из краскомета

- **Заправка**: Подойдите к резервуару своей команды и зажмите `E` для пополнения запаса краски

- **Боты**: 11 компьютерных игроков соперничают с вами

## 🚀 Запуск игры

### Вариант 1: Локальный сервер (рекомендуется)

```bash
# Перейдите в директорию проекта
cd splatoon-game

# Запустите локальный HTTP-сервер
# Если установлен Python 3:
python -m http.server 8000

# Или если установлен Node.js:
npx http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 2: Прямое открытие файла

Некоторые браузеры могут блокировать загрузку модулей при прямом открытии HTML-файла. Если игра не запускается, используйте вариант 1.

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперёд |
| `S` / `↓` | Движение назад |
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| Мышь | Поворот камеры |
| ЛКМ | Стрельба краской |
| `E` | Заправка у резервуара |

## ⚙️ Настройка баланса

Все параметры баланса находятся в файле `src/config.js`:

```javascript
const CONFIG = {
    playerSpeedNormal: 5.0,      // Базовая скорость
    playerSpeedFast: 7.5,        // Скорость по своей территории
    playerSpeedSlow: 2.5,        // Скорость по вражеской территории
    
    paintTankCapacity: 100,      // Ёмкость бака
    paintConsumptionPerShot: 5,  // Расход за выстрел
    refuelRate: 20,              // Скорость заправки
    
    matchDuration: 300,          // Длительность матча (секунды)
    
    // ... другие настройки
};
```

## 🎨 Технические требования

- **Браузер**: Chrome, Firefox, Edge последних версий
- **WebGL**: Требуется поддержка WebGL 2.0
- **Производительность**: ≥30 FPS на средних ПК

## 🔧 Технологии

- **Three.js** (v0.158.0) — 3D-графика через WebGL
- **HTML5 Canvas** — динамическая текстура арены
- **Vanilla JavaScript** — без дополнительных фреймворков

## 🎮 Игровой процесс

1. **Выберите команду** на стартовом экране
2. **Двигайтесь** по арене и **закрашивайте** территорию
3. **Следите за запасом краски** — возвращайтесь к резервуару для заправки
4. **Избегайте вражеской территории** — на ней вы двигаетесь медленно
5. **Победите** закрасив больше территории чем другие команды!

## 📊 Интерфейс

- **Индикатор краски** (слева внизу) — показывает остаток краски
- **Таймер матча** (сверху по центру) — оставшееся время
- **Счётчик территории** (справа вверху) — проценты покрытия команд
- **Индикатор заправки** (снизу по центру) — появляется при заправке
- **Подсветка ног** — зелёная (быстро), серая (нормально), красная (медленно)
