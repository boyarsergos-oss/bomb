# 3D Drone Simulator - Режим редактора уровней

## Запуск игры с сервером для сохранения в файл

### Способ 1: Node.js сервер (рекомендуется)

1. Убедитесь, что у вас установлен Node.js
2. Откройте терминал в папке с игрой
3. Запустите сервер:
   ```bash
   node server.js
   ```
4. Откройте браузер и перейдите по адресу: http://localhost:3000

### Способ 2: Без сервера (сохранение только в браузере)

Просто откройте файл `drone-game.html` в любом современном браузере.
Данные будут сохраняться в localStorage браузера.

## Как использовать редактор уровней

1. Нажмите кнопку **"🛠️ Режим редактора"** в правом верхнем углу
2. Используйте кнопки редактора:
   - **+ Добавить зону** - добавить новую посадочную зону (красный круг)
   - **+ Добавить куб** - добавить препятствие
   - **+ Добавить дом** - добавить 3D модель дома
   - **Выбрать зону/объект** - выбрать объект для редактирования
   - **- Удалить выбранное** - удалить выбранный объект
   - **💾 Сохранить уровень** - сохранить все изменения
   - **📂 Загрузить уровень** - загрузить сохранённый уровень
   - **Выйти из редактора** - вернуться в режим игры

3. Для перемещения объектов:
   - Кликните на объект для выбора
   - Зажмите Shift + используйте стрелки для точного перемещения
   - Или просто кликните в нужное место на земле

## Механика перехода на следующий уровень

- Каждая посадочная зона представляет отдельный уровень
- Когда вы создаёте несколько зон в редакторе, они автоматически становятся уровнями 1, 2, 3 и т.д.
- Чтобы перейти на следующий уровень:
  1. Выйдите из режима редактора
  2. Подлетите к красной зоне (уровень 1)
  3. Аккуратно приземлитесь внутри круга (высота < 1м, скорость < 0.5 м/с)
  4. После успешной посадки автоматически загрузится следующий уровень
  5. Дрон переместится на стартовую позицию, а следующая зона станет активной

## Файл сохранения

При использовании Node.js сервера данные уровня сохраняются в файл `level.json` в той же папке.
Этот файл можно:
- Редактировать вручную (в формате JSON)
- Передавать другим игрокам
- Резервировать и восстанавливать

Пример структуры файла level.json:
```json
{
  "landingZones": [
    {
      "position": { "x": 0, "y": 0, "z": -20 },
      "radius": 3,
      "landed": false
    }
  ],
  "houses": [
    {
      "position": { "x": 10, "y": 0, "z": 15 }
    }
  ]
}
```

## Управление дроном

### Клавиатура (ПК):
- W/S - Тяга вверх/вниз
- A/D - Поворот влево/вправо
- ↑/↓ - Наклон вперёд/назад
- ←/→ - Крен влево/вправо
- Пробел - Сброс позиции

### Сенсорное управление (мобильные):
- Левый джойстик - Тяга/Поворот
- Правый джойстик - Наклон/Крен
