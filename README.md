# Quantum Disarm - 50 Уровней Головоломок

## 🎮 Описание игры

**Quantum Disarm** - это уникальная 3D-головоломка где вы должны обезвредить 50 квантовых бомб чтобы спасти человечество!

## 🕹️ Как играть

1. Откройте `index.html` в любом современном браузере
2. Используйте **стрелки** или **WASD** для перемещения
3. Нажмите **R** для рестарта уровня

## 🎯 Элементы игры

| Элемент | Описание |
|---------|----------|
| 🟢 Зелёный круг | Игрок |
| 🔴 Красный квадрат | Бомба (цель) |
| ⬜ Серый квадрат | Стена |
| 🟡 Жёлтый квадрат | Переключатель |
| 🔵 Голубой квадрат | Лёд (скольжение) |
| 🟣 Бирюзовый портал | Телепортация |
| 🔴 Красный полупрозрачный | Лазер (опасно!) |
| 🚪 Красная дверь | Открывается переключателем |

## 🎮 Особенности

- **50 уникальных уровней** с нарастающей сложностью
- **6 различных механик**: стены, переключатели, лёд, порталы, лазеры, двери
- **Система времени и ходов** для отслеживания прогресса
- **Анимированная графика** с эффектами частиц
- **Постепенное обучение** - новые механики вводятся постепенно

## 🏆 Уровни сложности

- **Уровни 1-10**: Обучение основам
- **Уровни 11-20**: Средние головоломки
- **Уровни 21-30**: Сложные комбинации
- **Уровни 31-40**: Очень сложные
- **Уровни 41-50**: Экспертные испытания

## 💻 Запуск

Просто откройте файл `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`
